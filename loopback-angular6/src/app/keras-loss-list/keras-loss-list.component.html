Função Perda (loss) Keras

A função de perda em um modelo de deep learning serve para avaliar a diferença entre a saída prevista pelo modelo e o valor real dos dados de treinamento. Essa diferença é quantificada pela função de perda, que produz um valor numérico que representa o quão bem o modelo está performando em sua tarefa específica. A ideia é encontrar os valores dos pesos que minimizem a perda média em um conjunto de dados de treinamento, durante o processo de treinamento do modelo.
